# Vibe Vault for VS Code

Thin companion extension. Secret access flows through **vibevault-mcp**, not this extension.

## Setup

1. Install Vibe Vault on macOS
2. `vibevault mcp install --client vscode`
3. `vibevault skill install`

## Commands

- **Vibe Vault: Install MCP Config** — CLI install for MCP + skill
- **Vibe Vault: Open AI Agents Setup** — setup instructions

## Develop

```bash
cd extensions/vscode-vibevault && npm install && npm run compile
```
